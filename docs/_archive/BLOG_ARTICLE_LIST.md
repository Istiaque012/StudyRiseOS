## How to Use This List

- **Priority tiers:** P0 = publish first (highest search volume + direct product fit), P1 = next wave, P2 = long-tail / niche
- **Word count targets** are based on what currently ranks for each keyword cluster
- **Audience tags:** `[Exam]` = AMC/PLAB/USMLE candidates · `[Uni]` = university students · `[MBBS-BD]` = Bangladeshi medical students · `[All]` = all audiences
- Articles should link back to the relevant StudyRise mode landing page or feature where applicable
