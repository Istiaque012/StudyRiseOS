# BMDC Curriculum Reference

Authoritative BMDC MBBS Curriculum 2021 distilled. This is the correctness authority — the app must mirror these facts exactly.

## 4 Professional Phases
- Phase 1: 1300 total marks
- Phase 2: 600 total marks
- Phase 3: 900 total marks
- Phase 4: 1500 total marks

## Card/Item Structure
Per-subject breakdown (e.g., Anatomy 9 cards, Physiology 8). Each card contains items. Items have mark allocations across Written/SOE/Practical/Formative.

## Key Rules
- Pass mark: 60% per component separately
- GPA scale per Table 13
- Honours = A grade (75%), NOT 85%
- Phase-1 lockout: must pass Phase 1 before Phase 2
- 12-year completion window
- Per-class-type attendance requirements (Table 102)

## Integrated Teaching & Field Placements
Community Medicine (COME) and Forensic Medicine have field placement requirements. Integrated teaching sessions are cross-departmental.

## Internship
1-month joining period, UHC posting, 2-year completion window.
