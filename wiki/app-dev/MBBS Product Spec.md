# MBBS Product Spec

Complete product specification for the MBBS Bangladesh Mode redesign.

## Vision
Digital twin of the paper continuous-assessment card. Every BMDC student carries a physical card tracking their progress — this module digitizes it faithfully.

## 13 Feature Parts
Academic structure, gate system, onboarding, dashboard, items/cards, attendance, formatives, timetable, exam prep (viva + university question banks), clinical tracking, exam period system, pass probability projection, supplementary features.

## Eligibility Gates
Four BMDC gates a student must pass:
1. Attendance gate (per class type)
2. Formative assessment gate (10%)
3. Items/cards completion gate
4. Previous professional exam gate

## Onboarding
5-step overhaul that pre-loads real BMDC cards for the student's phase and subjects.

## Priority Order
R0–R10+ build sessions, each adding one feature layer. See [[MBBS Build Sessions]].

## Payment
bKash/Nagad for Bangladesh. Ethical-floor features (core tracking) always free.
