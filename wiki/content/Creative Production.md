# Creative Production

Production manual for creating social media visual assets. Contains AI image generation prompts, video production scripts, thumbnail briefs, and a brand-consistent prompt library. No face on camera. No stock photos. Everything brand-aligned. Companion to [[Social Content Formats]], [[Social Playbook]], and [[Brand Visual System]].

- Display: Plus Jakarta Sans Bold/ExtraBold (hero-level only)
- Body: Inter Regular
- Data/numbers: Inter Bold (tabular-nums)

### Logo
- Dark backgrounds: full-color (gradient icon + white text). Bottom-right default, always small. Never recolour/stretch/rotate.

## Quality Checklist (12 items)

Before publishing any asset: brand colours correct, Inter font, logo present, mobile-readable, no banned words, CTA language correct, no stock photos, facts verified with source visible, correct dimensions, [MBBS-BD] tone check, demo data in screenshots, file named correctly.
