# Brand Visual System

Version 2.3 of the StudyRise visual identity.

## Logo
4-bar ascending chevron mark. Files: studyrise-logo-full-color.svg, studyrise-logo-deep.svg.

## Color Palette
- **Navy** — primary dark
- **Blue** — primary brand
- **Purple** — accent
- **Cyan** — highlight/interactive

All colors defined as CSS custom properties in tokens.css. **No hardcoded hex values** — always use tokens.

## Typography
3-font stack (required on every page):
1. **Plus Jakarta Sans** — headings
2. **Inter** — body
3. **JetBrains Mono** — code/data

## Spacing
8px grid system.

## Components
Buttons, cards, chips, inputs, rings, tables — all spec'd in [[Brand Visual System]].

## Dark Mode
`data-theme="dark"` — OLED-friendly tokens. Full dark token set defined.

## Expressive Landing Layer
Ambient gradient glow, refined depth & glass surfaces — for marketing pages only.

## Key Rule
No hardcoded hex anywhere. Use CSS custom properties from tokens.css. See also [[Site Shell System]], [[Motion & Animation]].
